# Kabupaten Karangasem

## Ringkasan

Kabupaten Karangasem (ibu kota Amlapura) timur Bali, luas 839 km², populasi ~536 ribu jiwa.

## Sejarah singkat

- Kerajaan Karangasem (abad 17).

## Fun fact / hal menarik

- Gunung Agung, gunung tertinggi di Bali (3.031 m).

## Budaya

- Tari Balih.

## Kuliner

- Lawar Karangasem.

## Wisata

- Tirta Gangga, Amed (snorkeling).
