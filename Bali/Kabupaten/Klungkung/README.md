# Kabupaten Klungkung

## Ringkasan

Kabupaten Klungkung (ibu kota Semarapura) pusat sejarah Bali, luas 314 km², populasi ~223 ribu jiwa.

## Sejarah singkat

- Pusat Kerajaan Klungkung (Gelgel).

## Fun fact / hal menarik

- Kerta Gosa (pengadilan lukis).

## Budaya

- Pusat seni lukis.

## Kuliner

- Serapah Klungkung.

## Wisata

- Kerta Gosa, Puri Semarapura.
