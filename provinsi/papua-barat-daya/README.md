# Papua Barat Daya

## Ringkasan

Papua Barat Daya (PBD) ibukota Sorong, luas 35.000 km², populasi ~600rb, provinsi ke-38 (2022). [web:252][web:270]

## Administrasi

- 5 kabupaten: Maybrat, Raja Ampat, Sorong, Sorong Selatan, Tambrauw.
- 1 kota: Sorong. [web:262][web:270]
