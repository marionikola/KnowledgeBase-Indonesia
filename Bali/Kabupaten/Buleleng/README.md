# Kabupaten Buleleng

## Ringkasan

Kabupaten Buleleng (ibu kota Singaraja) utara Bali, luas 1.322 km², populasi ~826 ribu jiwa.

## Sejarah singkat

- Pusat Kerajaan Buleleng (abad 17).

## Fun fact / hal menarik

- Pantai terpanjang Bali.

## Budaya

- Tari Legong Keraton.

## Kuliner

- Ayam betutu Buleleng.

## Wisata

- Pantai Lovina (dolphin), Pura Meduwe Karang.
