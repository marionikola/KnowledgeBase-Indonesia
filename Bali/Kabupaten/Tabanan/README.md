# Kabupaten Tabanan

## Ringkasan

Kabupaten Tabanan (ibu kota Tabanan) barat daya Bali, luas 849 km², populasi ~476 ribu jiwa.

## Sejarah singkat

- Kerajaan Tabanan.

## Fun fact / hal menarik

- Sawah terluas di Bali; Jatiluwih masuk Warisan Dunia UNESCO.

## Budaya

- Subak Jatiluwih (UNESCO).

## Kuliner

- Lawar Tabanan.

## Wisata

- Tanah Lot, Jatiluwih.
