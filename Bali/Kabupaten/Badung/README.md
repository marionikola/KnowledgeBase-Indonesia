# Kabupaten Badung

## Ringkasan

Kabupaten Badung (ibu kota Mangupura) merupakan pusat pariwisata Bali selatan, luas 418 km², populasi ~550 ribu jiwa.

## Sejarah singkat

- Disebut sejak abad 17 dalam surat Gubernur Jenderal JP Coen (1619); dulunya pusat perdagangan budak.

## Fun fact / hal menarik

- Popularitas Kuta berawal dari pemakaman pedagang Denmark Viggo Drieberg (1930-an).

## Budaya

- Pusat tari Kecak dan surfing.

## Kuliner

- Babi guling khas Jimbaran.

## Wisata

- Pura Taman Ayun, Pantai Kuta, Uluwatu, Monumen Perjuangan Bangsal.
