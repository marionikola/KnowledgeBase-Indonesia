# Kabupaten Jembrana

## Ringkasan

Kabupaten Jembrana (ibu kota Negara) barat Bali, luas 841 km², populasi ~329 ribu jiwa.

## Sejarah singkat

- Bagian Kerajaan Blambangan.

## Fun fact / hal menarik

- Gerbang barat Bali.

## Budaya

- Pura Gede Luh.

## Kuliner

- Pepes ikan Jembrana.

## Wisata

- Pantai Balian, Pura Rambut Siwi.
