# Kalimantan Utara

## Ringkasan

Kalimantan Utara (Kaltara) ibukota Tanjung Selor, luas 75.467 km², populasi ~750rb, pemekaran Kaltim 2012. [web:201]

## Administrasi

- 4 kabupaten: Bulungan, Malinau, Nunukan, Tana Tidung.
- 1 kota: Tarakan. [web:201]

## Sejarah
