# Jambi

## Ringkasan

Jambi adalah provinsi di Sumatera bagian tengah, ibu kota Jambi, luas 50.058 km², populasi ~3,6 juta jiwa.

## Administrasi

**Kabupaten (9):**
- Batanghari
- Bungo
- Kerinci
- Merangin
- Muaro Jambi
- Sarolangun
- Tanjung Jabung Barat
- Tanjung Jabung Timur
- Tebo

**Kota (2):**
- Jambi
- Sungai Penuh

Lihat `kabupaten/` dan `kota/`.

## Sejarah singkat

- Kerajaan Melayu Jambi (abad 7); Candi Muaro Jambi (abad 7-13).

## Fun fact / hal menarik

- Kompleks Candi Muaro Jambi, situs Buddha terluas di Asia Tenggara.
- Gunung Kerinci, gunung berapi tertinggi di Indonesia (3.805 m).

## Budaya

- Tari Sekapur Sirih; Batik Jambi; Rumah adat Kajang Lako.

## Kuliner

- Tempoyak (fermentasi durian); Gulai ikan patin; Nasi gemuk.

## Wisata

- Candi Muaro Jambi, Taman Nasional Kerinci Seblat, Danau Kerinci.

## Tokoh

- Sultan Thaha Syaifuddin (pahlawan nasional).
