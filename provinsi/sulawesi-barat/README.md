# Sulawesi Barat

## Ringkasan

Sulawesi Barat adalah provinsi di barat Pulau Sulawesi, ibu kota Mamuju, luas 16.787 km², populasi ~1,4 juta jiwa. Dibentuk tahun 2004.

## Administrasi
