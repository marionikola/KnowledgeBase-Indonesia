# Kabupaten Gianyar

## Ringkasan

Kabupaten Gianyar (ibu kota Gianyar) pusat seni Bali, luas 364 km², populasi ~508 ribu jiwa.

## Sejarah singkat

- Kerajaan Gianyar (pemekaran Klungkung 1906).

## Fun fact / hal menarik

- 7 kecamatan seni ukir.

## Budaya

- Pusat tari/wayang.

## Kuliner

- Sate lilit Gianyar.

## Wisata

- Ubud (monkey forest, pasar seni), Tegallalang.
