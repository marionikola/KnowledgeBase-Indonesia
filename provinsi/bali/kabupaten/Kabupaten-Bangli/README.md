# Kabupaten Bangli

## Ringkasan

Kabupaten Bangli (ibu kota Bangli) satu-satunya kabupaten Bali tanpa pantai laut, luas 520 km², populasi ~262 ribu jiwa.

## Sejarah singkat

- Bagian Kerajaan Gelgel; terkenal Danau Batur.

## Fun fact / hal menarik

- Terkurung daratan, batas 5 kabupaten lain.

## Budaya

- Pusat pura dan sawah.

## Kuliner

- Babi guling Bangli.

## Wisata

- Danau Batur Kintamani, Pura Kehen.
