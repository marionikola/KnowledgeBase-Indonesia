# Kota Denpasar

## Ringkasan

Kota Denpasar ibu kota Provinsi Bali, luas 111 km², populasi ~726 ribu jiwa.

## Sejarah singkat

- Pusat Puputan Badung (1906).

## Funfact / hal menarik

- Kota paling padat Bali.

## Budaya

- Sanur Festival.

## Kuliner

- Tipat bali.

## Wisata

- Bajra Sandhi, Pasar Badung.
